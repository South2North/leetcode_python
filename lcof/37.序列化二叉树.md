[剑指 Offer 37. 序列化二叉树](https://leetcode-cn.com/problems/xu-lie-hua-er-cha-shu-lcof/)

请实现两个函数，分别用来序列化和反序列化二叉树。

示例: 
```sh
你可以将以下二叉树：

    1
   / \
  2   3
     / \
    4   5

序列化为 "[1,2,3,null,null,4,5]"
```

注意：本题与主站 297 题相同：https://leetcode-cn.com/problems/serialize-and-deserialize-binary-tree/

代码：
```python
'''
思路：序列化时使用先序遍历，当节点为None时，插入None
     反序列化时维护一个idx，表示进行到当前的值
     注意：这里不能写成这样：
        root.left=helper(data,idx+1)
        root.right=helper(data,idx+2)
        假设root为1，left为2，right为3，那么下一次调用就变成了了root为2，left为3，显然不正确
        而应该将idx作为外部变量，确保每一次idx+1都生效了（data索引往后移了，不然就会重复）
'''
# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Codec:

    def serialize(self, root):
        """Encodes a tree to a single string.
        
        :type root: TreeNode
        :rtype: str
        """
        res=[]
        def helper(root):
            if not root:
                res.append(None)
                return
            res.append(root.val)
            helper(root.left)
            helper(root.right)
        helper(root)
        res=[str(x) for x in res]
        return ','.join(res)
        

    def deserialize(self, data):
        """Decodes your encoded data to tree.
        
        :type data: str
        :rtype: TreeNode
        """
        data=list(data.split(','))
        idx=0
        n=len(data)
        def helper():
            nonlocal idx
            if idx>=n:
                return
            if data[idx]=='None':
                return
            root=TreeNode(int(data[idx]))
            idx+=1
            root.left=helper()
            idx+=1
            root.right=helper()
            return root
        return helper()
```
